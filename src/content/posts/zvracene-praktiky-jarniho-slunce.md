Zvrácené praktiky jarního slunce \
ovíjí kotníky rozverných dívek \
lechtá je v zátylku \
hřeje jim tváře \
ale už za chvilku \
skrývá se v páře \
mraků \
drzé a bez rozpaků

Zvrácené praktiky jarního slunce \
neposedné, přelétavé, nestranné \
vysvitne, navnadí, zmizí \
jednou je blízké \
jindy je cizí \
pohladí \
kopne \
jen jizva zůstane

Zvrácené praktiky jarního slunce \
dělá si \
co jej zapadne \
a ani se nerozloučí
